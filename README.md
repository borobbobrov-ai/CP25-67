# Currency Converter - Приложение для конвертации валют

## Автор
**Берсенев Станислав** 

## Описание
GUI-приложение для конвертации валют с использованием внешнего API, сохранением истории конвертаций в JSON и возможностью загрузки истории.

## Как получить API-ключ
1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com)
2. Нажмите "Get Free API Key"
3. Введите email и подтвердите регистрацию
4. Получите бесплатный API-ключ (до 1500 запросов в месяц)
5. Вставьте ключ в код: `self.api_key = "ваш_api_ключ"`
